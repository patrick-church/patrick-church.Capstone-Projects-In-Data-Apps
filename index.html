Automated Cross-Validation for LWR and Random Forest

My code displays a class that implements a Locally Weighted Regression (LWR), a regression technique that emphasizes local data points when making predictions. It also offers the option to compare this method with Random Forest by switching the model type.

The cross_validate function in the class incorporates all the other functions from the class for optimal automation. It brings together the fit, predict, mse, and r2_score functions to automate the K-fold cross-validation for Locally Weighted Regression (LWR) and Random Forest.

Kernel Functions

Before we get into the functions within the class, there are four kernel functions outside of the class that help create the weights for LWR:

  • Gaussian
  • Tricubic
  • Epanechnikov
  • Quartic

The different kernel functions allow the user to test and choose the most accurate one.

PatricksModel Class Constructor

The constructor accepts parameters like model type ('lwr' or 'rf'), kernel type, neighborhood fraction (f), number of iterations, and whether to include an intercept.

Model Fitting

For LWR, the input features are scaled using MinMaxScaler. For Random Forest, the fit method is used from RandomForestRegressor.

Weight Calculation

This function calculates the weights for each training point relative to the new data point, applying the selected kernel function.

Prediction

For LWR, predictions are made by computing weights and solving for the regression coefficients via weighted least squares. For Random Forest, the predict function generates predictions from the trained model.

Model Evaluation

  • mse: Calculates the Mean Squared Error (MSE).
  • r2_score: Computes the R² score.

Cross-Validate Function

The cross_validate function automates evaluation by splitting the dataset into K-folds, training the model, generating predictions, and calculating performance metrics (MSE and R² score) for each fold. It automates the entire process, making it fast and convenient to evaluate models.

Examples:

Cross-Validation for Locally Weighted Regression (LWR)


lwr_model = PatricksModel(model_type='lwr', kernel='tricubic', f=1/3, iter=3, intercept=True)

Cross-Validation for Random Forest


rf_model = PatricksModel(model_type='rf', rf_params={'n_estimators': 100, 'max_depth': 10})
